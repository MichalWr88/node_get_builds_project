# node_get_builds_project
# [v1.2.0](https://github.com/MichalWr88/node_get_builds_project/releases/tag/1.2.0)
# QUICK START

1. install dependencies
```javascript 
  npm i
```
2.  set credentials 
```javascript 
  npm run auth
```
3. build applications 
```javascript 
  npm run setBuild
``` 
4. download build
```javascript 
  npm run getDownload
```
